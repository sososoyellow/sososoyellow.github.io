# Morandi Caramel Design System

## Color Palette
- **Background**: #FDFAF4 (warm off-white) - Main page background.
- **Surface**: #F8F0E0 - Card backgrounds and secondary surfaces.
- **Primary**: #F4E3BB (cream yellow) - Brand anchor color.
- **Accent Warm**: #C2A46A (caramel milk tea) - Decorative elements and accents.
- **Accent Dark**: #8B5A2B (dark caramel) - Interactive elements and navigation links.
- **Text Primary**: #2F1B0C (deep brown) - Headings and high-contrast identity text.
- **Text Body**: #4A4A4A - Standard body copy.
- **Text Muted**: #C2A46A - Annotations and secondary labels.
- **Border**: rgba(194, 164, 106, 0.25) - Subtle separators.

## Typography
- **Headings**: System Sans-Serif (Heiti), Bold/Medium.
- **Body/UI**: Arial, Regular.
- **Identity**: Large font sizes for "Edith" and key attributes.
